*********************
Fluid Data Generation
*********************

*SU2 DataMiner* uses a configuration class in order to store important information regarding the data generation, data mining, and 
manifold generation processes. This page lists some of the important functions of the *Config* class which acts as the :ref:`base<base_dg>` 
class for configurations specific to the application such as :ref:`NICFD<NICFD_dg>` and FGM :ref:`FGM<FGM_dg>`, for which additional settings can be specified.
