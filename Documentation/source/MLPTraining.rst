MLP Training
************

*SU2 DataMiner* uses a configuration class in order to store important information regarding the data generation, data mining, and 
manifold generation processes. This page lists some of the important functions of the *Config* class which acts as the :ref:`base<base_mlp>` 
class for configurations specific to the application such as :ref:`NICFD<NICFD_mlp>` and FGM :ref:`FGM<FGM_mlp>`, for which additional settings can be specified.
